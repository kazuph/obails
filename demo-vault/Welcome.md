# Welcome to Obails

Obails is a lightweight Obsidian alternative built with modern technologies.

## Features

- **Fast & Native** - Built with Go backend
- **12 Beautiful Themes** - Light and dark options
- **Mermaid Support** - Diagrams rendered inline
