# Getting Started

## Installation

1. Download from GitHub Releases
2. Move to Applications
3. Right-click → Open

## Configuration

Edit `~/.config/obails/config.toml`:

```toml
[vault]
  path = "/path/to/your/vault"
```

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| ⌘, | Open settings |
| ⌘S | Save file |
