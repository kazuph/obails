# Welcome to Obails

A lightweight, fast markdown editor built with modern technologies.

## Features

- **Live Preview** - See your changes in real-time
- **Multiple Themes** - Choose from 12 beautiful themes
- **Mermaid Diagrams** - Create flowcharts and diagrams
- **Wiki Links** - Connect your notes with [[Features]]

## Quick Start

1. Select a note from the sidebar
2. Start typing in the editor
3. Preview updates automatically
