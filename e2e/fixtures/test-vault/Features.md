# Features

Obails comes packed with powerful features for your note-taking needs.

## Editor

The editor supports:
- Full **Markdown** syntax
- *Italic*, **bold**, and ~~strikethrough~~
- Code blocks with syntax highlighting
- Task lists like this:
  - [x] Completed task
  - [ ] Pending task

## Navigation

### File Tree
Browse your vault with the sidebar file tree.

### Outline Panel
Jump to any heading with the outline panel on the right.

### Backlinks
See which notes link to the current note.
